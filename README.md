# Project Description

Project Title: My Portfolio

Overview :
Developed a responsive personal portfolio website to showcase my skills, academic projects, and professional information.
The website provides an interactive and user-friendly interface, allowing visitors to view projects and contact me easily.
The design focuses clean UI, smooth navigations, and mobile responsiveness.

Purpose:
The main purpose of a portfolio is to showcase my skills, projects and other details available in one place to the visitors. It act as a digital resume that helps recruiters, clients, and companies to understand my technical ablities and
knowledge.

Technologies Used:
1.HTML
2.CSS
3.React

Key Points:
Responsive design for mobile, tablet, and desktop devices.
Navigations using React Router.
Home section with hero banner and basic introduction.
Project section displays project details with image.
Skills section highlighting my technical skills through progress bar.
About section shows introduction of myself.
Contact form with basic validations.

Visuals:
Project related images.

# How to run the project

First Step : Open the project folder.
Second Step : run command npm run dev.
Third Step : Click on the link then the project is shown.
